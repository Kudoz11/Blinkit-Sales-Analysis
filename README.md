# Blinkit Sales Analysis 🛒

### Project Objective
The goal of this project is to perform a comprehensive sales analysis for Blinkit to identify key performance indicators (KPIs) and provide actionable insights into customer preferences and outlet performance using Power BI, SQL, and Python.

### Tech Stack
* **Data Cleaning:** SQL, Python (Pandas)
* **Visualization:** Power BI
* **Analysis:** Exploratory Data Analysis (EDA)

### Key Insights
* **Total Sales:** Identified high-performing outlet types and top-selling categories.
* **Customer Behavior:** Analyzed impact of item fat content on sales.
* **Outlet Performance:** Evaluated sales across different city tiers.
